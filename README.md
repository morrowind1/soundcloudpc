================================
  soundcloud desktop
  инструкция по сборке .exe
================================
 
 
шаг 1 — скачай python
──────────────────────
1. зайди на сайт: https://www.python.org/downloads/
2. нажми большую кнопку "download python 3.12.x"
3. запусти скачанный файл
4. !! обязательно поставь галочку "add python to path" внизу окна
5. нажми "install now"
6. дождись установки и закрой окно
 
 
шаг 2 — скачай inno setup
──────────────────────────
1. зайди на сайт: https://jrsoftware.org/isdl.php
2. скачай "inno setup 6.x.x"
3. запусти и установи (просто жми далее везде)
 
 
шаг 3 — подготовь файлы
────────────────────────
создай где угодно папку, например: c:\soundcloud\
скопируй в неё эти файлы из репозитория:
 
  soundcloud_app.py
  build.bat
  installer.iss
 
 
шаг 4 — собери exe
───────────────────
1. открой папку c:\soundcloud\
2. зажми shift и кликни правой кнопкой на пустом месте
3. выбери "открыть окно powershell здесь"
   (или "открыть терминал")
4. вставь команду и нажми enter:
 
   pip install PyQt6 PyQt6-WebEngine pyinstaller
 
5. дождись установки (может занять 2-5 минут)
6. теперь запусти build.bat — кликни на него правой кнопкой
   и выбери "запуск от имени администратора"
7. появится чёрное окно — жди пока не закроется само
 
 
шаг 5 — готово
───────────────
после того как build.bat завершится в папке появятся:
 
  dist\SoundCloud.exe       ← просто exe без установки
  Output\SoundCloud_Setup.exe  ← полноценный установщик ✓
 
SoundCloud_Setup.exe можно кидать на любой компьютер —
python и библиотеки устанавливать не нужно, всё уже внутри
 
 
================================
  если что-то пошло не так
================================
 
— pip не найден
  переустанови python с галочкой "add python to path"
 
— build.bat выдаёт ошибку
  убедись что все три файла лежат в одной папке
 
— inno setup не найден / установщик не собрался
  установи inno setup (шаг 2) и запусти build.bat снова
 
— антивирус ругается на exe
  это нормально для файлов собранных pyinstaller
  добавь папку в исключения антивируса
